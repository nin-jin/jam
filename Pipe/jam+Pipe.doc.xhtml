$jam_Pipe

Выстраивает переданные ей функции в один трубопровод, где результат исполнения предыдущей подаётся на вход следующей.

_test.ok( $jam_Pipe ) _test.ok( $jam_Pipe() ) _test.ok( $jam_Pipe()( 123 ), 123 ) _test.ok( $jam_Pipe( Math.abs ), Math.abs ) _test.ok( $jam_Pipe( Math.abs )( -5 ), 5 ) _test.ok( $jam_Pipe( parseInt, Math.abs )( '-5px' ), 5 )
