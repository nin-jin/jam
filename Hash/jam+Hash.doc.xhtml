$jam_Hash

Создаёт хэш-таблицу. Если в параметрах передать объект и префикс - будет использовать его для хранения данных.

_test.ok( $jam_Hash ) _test.not( $jam_Hash().has( 'foo' ) ) _test.not( $jam_Hash().has( 'hasOwnProperty' ) ) var hash= $jam_Hash() hash.put( 'foo', 'bar' ) _test.ok( hash.has( 'foo' ) ) var hash= $jam_Hash() hash.put( 'foo', 'bar' ) _test.ok( hash.get( 'foo' ), 'bar' ) var hash= $jam_Hash({ prefix: 'foo:', obj: { 'foo:bar': 123 } }) _test.ok( hash.has( 'bar' ) ) var hash= $jam_Hash({ prefix: '', obj: $jam }) _test.ok( hash.has( 'Hash' ) )
