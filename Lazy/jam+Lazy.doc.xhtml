$jam_Lazy

Создаёт ленивую функцию. Принимает на вход фабрику, которую использует для создания актуальной функции при первом вызове.

_test.ok( $jam_Lazy ) var lazy= $jam_Lazy( function(){ return function( a, b ){ return b } }) _test.ok( lazy( 1, 2 ), 2 ) var lazy= $jam_Lazy( function(){ var a= Math.random() return $jam_Value( a ) }) _test.ok( lazy(), lazy() )
